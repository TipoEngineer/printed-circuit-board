Command Git:
1 Команда: git clone ссылка //склонировать все файлы с репозиторя если есть ссылка 
2 Команда: git status //Проверить статус фалов
3 Команда: git add -A // приготовить в репозиторий на облако
4 Команда: git commit -m "Сообщение"// приготовить в репозиторий на облако и добавить сообщение об обновлении
5 Команда: git push // Отправить файл
6 Команда: git pull // получить все файлы
